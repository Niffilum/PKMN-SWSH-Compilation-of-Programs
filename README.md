# Auto Breeding

Prerequisites:
- Route 5 Nursery
- Cursor in menu must be in map
- Have 6 pokemon in party
- Text speed: Fast
- Nickname:NO
- Box: Manual

Note: Poke Menu must be closed and you are in the overworld
Plug it in on the Change Grip/Order Menu in switch Home
